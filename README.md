# 专业培养方案识别可视化

这是一个本地运行的静态网站，用于上传或粘贴专业培养方案文本，并自动识别课程、学分、学期、课程类别，生成可视化看板。

## 使用方式

直接打开 `index.html`，或把文件夹部署到任意静态站点服务。

当前支持：

- 首页介绍、导航、上传工作台、功能说明、示例格式和页脚
- 上传 `.txt`、`.csv`、`.md`、`.json` 文本类文件
- 粘贴培养方案正文
- 自动识别课程名称、学分、学期、课程类别
- 展示总课程数、总学分、学期范围、识别完整度
- 展示学分结构环图、学期路线图、课程清单
- 搜索与按课程类别筛选
- 导出识别结果 JSON

## 推荐输入格式

```text
课程名称 学分 学期 课程类别
高等数学A 5 第1学期 公共基础课
程序设计基础 4 第1学期 专业基础课
数据结构 4 第2学期 专业核心课
毕业设计 8 第8学期 毕业设计
```

CSV 或 JSON 也可以使用 `课程名称`、`学分`、`学期`、`课程类别` 等字段。

## 后续可扩展

- 接入后端后，可增加 PDF、Word 的正文提取
- 接入 OCR 后，可识别扫描版培养方案
- 可增加课程先修关系图、毕业要求矩阵、学分达成校验
